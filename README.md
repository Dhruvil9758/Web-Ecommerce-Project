### Database Schema Design

**Products Schema (MongoDB)**

- `product_id` : Number
- `name`: String
- `description`: String
- `price`: Number
- `model`: String

**Users Schema (MongoDB)**

- `username`: String
- `password`: String
- `email`: String

**Order Schema (MongoDB)**

- `OrderID`: String
- `TotalAmount`: Number
- `OrderDate`: Number
- `OrderStatus`: String
